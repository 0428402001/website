Stable Release

The latest stable release of ZMap is version 1.4.0 which can be download at zmap.0.4.10.tar.gz. Any issues with the latest release should be reported on the GitHub Issue Tracker.


Bleeding Edge

The latest zmap code can be downloaded at https://github.com/zmap/zmap/tarball/master or cloned from the git repository git://github.com/zmap/zmap.git. ZMap utilizes git submodules for some libraries. As such, you will need to recursively clone the repository or run git submodule init and git submodule update after cloning.


Building and Installing ZMap

ZMap was designed to run on GNU/Linux based systems and can be built with most recent versions of gcc. ZMap requires GMP, a free library for arbitrary precision arithmetic and libpcap. GMP can installed on Debian based systems by running sudo apt-get install libgmp3c2 libgmp3-dev or sudo yum install gmp gmp-devel on RHEL and Fedora based systems. LIBPCAP can be installed by sudo apt-get install libpcap0.8-dev on Ubuntu/Debian and sudo yum install libpcap-devel on Fedora/RHEL. If you are building ZMap with Redis support, you will need to install Hiredis.

Once these requirements have been installed, ZMap can be installed by running make and make install. ZMap does not build with Redis support by default. If you are building ZMap from source, you can build with Redis support by running make REDIS=true.
