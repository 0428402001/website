Building and Installing ZMap

ZMap was designed to run on GNU/Linux based systems and can be built with most recent versions of gcc. ZMap also requires GMP, a free library for arbitrary precision arithmetic. GMP can installed on Debian based systems by running sudo apt-get install libgmp3c2 libgmp3-dev or sudo yum install gmp gmp-devel on RHEL and Fedora based systems. If you are building ZMap with Redis support, you will need to install Hiredis.

Once these requirements have been installed, ZMap can be installed by running make and make install. ZMap does not build with Redis support by default. If you are building ZMap from source, you can build with Redis support by running make REDIS=true.
