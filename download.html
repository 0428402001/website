Downloading ZMap

The latest stable release of ZMap is version 1.4.0, which can be downloaded at zmap.0.4.10.tar.gz.

The latest bleeding-edge code can be downloaded at https://github.com/zmap/zmap/tarball/master or cloned from the git repository git://github.com/zmap/zmap.git.

Issues should be reported on the Github Issue Tracker or sent to our team at zmap-team@umich.edu.

Building and Installing ZMap

ZMap is developed for use on Linux and can be built with most recent versions of gcc. ZMap requires GMP and libpcap. Hiredis is also required for Redis support. These can be downloaded as follows:

Debian or Ubuntu: sudo apt-get install libgmp10 libgmp3-dev libpcap0.8-dev

Fedora, Red Hat Enterprise Linux, or CentOS: sudo yum install gmp gmp-devel libpcap-devel

Once these requirements have been installed, ZMap can be installed by running make and sudo make install.
